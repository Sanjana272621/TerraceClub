  # TerraceClub

## Overview
TerraceClub is a MERN stack web application for managing community terrace gardening. It allows residents to view garden spaces, manage plots, plan crops, track tasks, and post updates.

## Features
- User authentication (JWT based)
- Terrace and plot management
- Crop planning and maintenance tasks
- Activity log for updates, images, and notes
- Search and filters for plants, plots, and users
- Responsive UI for mobile and desktop

## Tech Stack

### Frontend
- React
- React Router
- Axios
- TailwindCSS (or your chosen CSS framework)

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JSON Web Tokens
- dotenv

## Setup

### Clone the repo
```bash
git clone https://github.com/<your-username>/terraceclub.git
cd terraceclub 
```

### Install Backend Dependencies
```bash
cd backend
npm install
```

### Install Frontend Dependencies
```bash
cd ../frontend
npm install
```

### Start Backend 
```bash
cd backend
npm run dev
```

### Start Frontend
```bash
cd frontend
npm run dev
```
